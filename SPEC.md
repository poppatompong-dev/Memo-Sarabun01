# SPEC: บันทึกข้อความ — ค่าที่ตรวจสอบกับแหล่งอ้างอิงจริง

ทุกค่าใน `app/preview/[id]/print.css` ต้องอ้างอิงแหล่งใดแหล่งหนึ่งดังนี้:

- **TEMPLATE** = `templates/แบบบันทึกข้อความ.doc` (template จริงที่เทศบาลใช้) — **น้ำหนักสูงสุด** เพราะคือสิ่งที่เจ้าหน้าที่ใช้ทำงานจริง
- **PDF 1** = `templates/1663663141_Article.pdf` (KM งานสารบรรณ — น.ส.ภวิตรา พวงนาค)
- **PDF 2** = `templates/ความรู้ด้านเทคนิคการเขียนหนังสือราชการ.pdf`
- **PDF 3** = `templates/8753t2.pdf` (คู่มือการเขียนหนังสือราชการ)
- **PDF 4** = `templates/ความรู้เพิ่มเติม.pdf` (scan ภาพ — OCR สำเร็จเป็น `templates/ความรู้เพิ่มเติม.txt`)
- **ระเบียบ** = ระเบียบสำนักนายกรัฐมนตรีว่าด้วยงานสารบรรณ พ.ศ. ๒๕๒๖ + ฉ.๒ พ.ศ. ๒๕๔๘ + ฉ.๔ พ.ศ. ๒๕๖๔

## คำลงท้าย

- บันทึกข้อความ**ไม่มีคำลงท้าย** (ขอแสดงความนับถือ ฯลฯ) — PDF 1 + PDF 3 ยืนยัน

## AI Prompt — กฎสำนวนที่ยังไม่ครอบคลุม (จาก PDF 2 หน้า 9-10)

ต้องเพิ่มในสำนวนการแก้ภาษา:

### คำเบา-คำหนักแน่น
- จะ (ทั่วไป) vs จัก (หนักแน่น คำขู่/คำสั่ง/คำกำชับ)
- ควร (แนะนำ จิตใจ) / พึง (วางมาตรฐาน สังคม) / ย่อม (บังคับ ไม่เด็ดขาด ใช้ดุลพินิจ) / ต้อง / ให้ (บังคับเด็ดขาด)

### คำบังคับ vs คำขอร้อง
- บังคับ: "ขอให้ส่ง / ให้ไปส่ง / ขอให้นำเสนอต่อไป"
- ขอร้อง: "โปรดส่ง / โปรดไปติดต่อ / โปรดนำเสนอต่อไป"

### คำทำลาย vs คำเสริมสร้าง
- ทำลาย: "ไม่ตั้งใจจึงสอบตก" / "โครงการที่ท่านเสนอใช้ไม่ได้" / "ท่านเข้าใจผิด"
- เสริมสร้าง: "ถ้าตั้งใจคงสอบได้" / "โครงการนับว่าดีแต่เกรงว่าจะยังทำไม่ได้" / "ความเข้าใจของท่านยังคลาดเคลื่อน"

### หลีกเลี่ยงคำเชื่อมซ้ำ
- ที่ / ซึ่ง / อัน
- และ / กับ / รวมทั้ง / ตลอดจน
- ใช้คำเชื่อมเดียวกันต่อกันหลายครั้ง → ใส่เฉพาะคำสุดท้าย

### คำใช้แทนกันได้/ไม่ได้
- กับ (ติดกัน เท่ากัน) / แก่ (สำหรับ ให้) / แด่ (ถวาย อุทิศ) / ต่อ (กระทำต่อฝ่ายเดียว)
- และ (ทั้งหมด) / หรือ (อย่างไหนก็ได้) / และหรือ (ทั้งสองก็ได้)

### สำนวนปิดท้ายตามวัตถุประสงค์ (PDF 2 หน้า 14-15) — **ต้องเพิ่มในคลังคำลงท้าย**
- คำซักซ้อม: "จึงขอเรียนซ้อมความเข้าใจมาเพื่อถือเป็นหลักปฏิบัติต่อไป"
- คำชี้แจง: "จึงเรียนชี้แจงมาเพื่อทราบ"
- คำยืนยัน: "จึงเรียนยืนยันมาเพื่อทราบ" / "จึงขอเรียนยืนยันข้อตกลงดังกล่าวมา ณ ที่นี้"
- คำเตือน: "บัดนี้ เวลาล่วงเลยมานานแล้ว ยังไม่ได้รับรายงาน... จึงขอเรียนเตือนมา"
- คำกำชับ: "จึงเรียนมาเพื่อจักได้ปฏิบัติตาม...โดยเคร่งครัดต่อไป"
- คำถาม: "จึงเรียนมาเพื่อขอทราบว่า..."
- คำหารือ: "จึงเรียนหารือมาว่า..."
